Initial commit: Add EMBI Scheduler C++ Simulator and Documentation

This commit establishes the foundational structure for the EMBI (Estimated Marginal Blocking Impact) Scheduler research repository.

Includes:
- Mathematical theory overview and project description (README.md)
- C++ discrete-event simulator for evaluating EMBI against MaxWeight, cmu, RR, and FCFS across various workloads
- Top-level documentation for architecture and experiments (docs/)
- Example bash scripts to run individual simulations and parameter sweeps (examples/)
- Contributing guidelines (CONTRIBUTING.md)
- Comprehensive .gitignore filtering build artifacts, python cache, and IDE configs
